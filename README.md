# MelodyQuest
Battle with music theory!

Requirements:
- A MIDI keyboard connected to your computer to play.

Disclaimer:
- This game was only tested on the latest version of Google Chrome.
- The game is unfinished.

Made at Hyphen-Hacks 2019!
